# Lost and Found - UWI CaveHill

This project is a full-stack web application designed for the University of the West Indies, Cave Hill Campus. It provides a platform for students and staff to report and search for lost and found items on campus.

## Features

-   **User Profiles**: Create and manage user accounts.
-   **Report Lost Items**: Submit details about items that have been lost.
-   **Report Found Items**: Submit details about items that have been found.
-   **Commenting System**: Engage in discussions on item reports.

## Technology Stack

-   **Frontend**: React.js (UI Library) and Next.js (Framework for UI and API routes).
-   **Backend**: Next.js API Routes.
-   **Database**: MongoDB (NoSQL Database).

## Getting Started

To get a local copy up and running, follow these simple steps:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/CSS-CaveHill-Campus/LostAndFound
    cd LostAndFound
    ```
2.  **Install dependencies**:
    ```bash
    bun install
    ```
3.  **Set up environment variables**: Create a `.env` file based on `.env.example` and configure your MongoDB connection string and any other necessary variables. (Will be discussed during the session)
4.  **Run the development server**:
    ```bash
    bun run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
